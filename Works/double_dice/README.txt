Double Dice
-----------

By: Honglu Xu

----
Info
----

It is a small work for rolling two dices. Have fun with it!

---------
User Case
---------

Please use any web brownser to open the "dice2.html" file.

Click on the first dice to roll the first dice.
Click on the second dice to roll the second dice.
Click on the "Click to rotate both dices" button to roll two dices together.
