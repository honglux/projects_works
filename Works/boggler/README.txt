Boggler
-------

By: Honglu Xu

----
Info
----

A boggler tool for a 4*4 table.

---------------
Compile and run
---------------

In the directory "boggler", Use the command "python3 boggler.py"

---------------------------
Executable file for windows
---------------------------

Unzip the file "bollger_windows.zip".

Excute the file "boggler.exe".

-----
Usage
-----

Enter a 16 charaters string as the boggler table.

Enter the 'e' to exit, anything else to continue.
