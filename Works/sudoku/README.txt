Sudoku
------

By: Honglu Xu

----
Info
----

A tool for solving sudoku problems

---------------
Compile and run
---------------

In the directory "sudoku", Use the command "python3 sudoku.py"

---------------------------
Executable file for windows
---------------------------

Unzip the file "sudoku_windows.zip".

Excute the file "sudoku.exe".

-----
Usage
-----

Enter any key to close the program.

-----
Board
-----

Board file: "myboard.txt"

'.' for the unknow number
