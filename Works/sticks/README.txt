Sticks
------

By: Honglu Xu

----
Info
----

A stick game with simple AI.

---------------
Compile and run
---------------

In the directory "sticks", Use the command "gcc -std=c99 sticks.c -o sticks"

Run "sticks"

---------------------------
Executable file for windows
---------------------------

Unzip the file "sticks_windows.zip".

Excute the file "sticks.exe".

-----
Usage
-----

Rules
-----
First decide how many sticks there are on the table.
Players will have to take sticks each turn, who takes the last stick will lose the game.

Enter the 'e' to exit, anything else to continue.
