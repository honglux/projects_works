Maze Solver
-----------

By: Honglu Xu

----
Info
----

A program to solve a maze problem by using the right hand rule.

---------------
Compile and run
---------------

Linux system:
In the directory "maze_solver", Use the command "make"

Run the program: "./maze_solver.out"

---------------------------
Executable file for windows
---------------------------

Unzip the file "maze_solver_windows.zip".

Excute the file "maze_solver.exe".

-----
Usage
-----

Press the "Enter" key to continue the progress.

Record
------
1. Use the command "./maze_solver.out >> xxxx.txt" instead of "./maze_solver.out", replace the "xxxx" to a custom file name you like.
2. Press the "Enter" key until the program finished.

Map-editing
-----------
The map file is "input.txt". Edit it to edit the map.
1. '@' represents for the wall.
2. 'x' (Lower case!) represents for the start posision. (Please put the start position on the edge of the map)
3. '.' represents for the empty place. Leave an empty place on the edge of the map to represent the exit.
