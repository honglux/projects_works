Calendar
--------

By: Honglu Xu

----
Info
----

A tool to check the date and day with the specific month and year, with the layout of the calendar for that month.

---------------
Compile and run
---------------

In the directory "Calendar": Use the command "python3 Calendar_layout.py"

---------------------------
Executable file for windows
---------------------------

Unzip the file "Calendar_windows.zip".

Excute the file "find_word.exe".

---------
Usage
---------

Enter the year number you want to check.

Enter the month number you want to check.

Type in 'e' with "Enter" to close the window, anyting else to do it again.
