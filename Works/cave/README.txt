Cave
----

By: Honglu Xu

----
Info
----

Trying to use the flood method to find out the number of closed chambers in a "cave".

---------------
Compile and run
---------------

In the directory "cave", Use the command "python3 cavern.py"

---------------------------
Executable file for windows
---------------------------

Unzip the file "cave_windows.zip".

Excute the file "caven.exe".

-----
Usage
-----

Use the 'Enter' to get to next step.

---
Map
---

The map list is the 'settings.txt' file.

Edit map
--------
Please use the same format to edit the map file
'cave' as the size of the cave: n row, n column
'hwall' as the horizontal wall, from (x,y) and extending n to the right side
'vwall' as the vertical wall from (x,y) and extending n to the down side

New map: Enter another line with the file name in the 'settings.txt' file.
